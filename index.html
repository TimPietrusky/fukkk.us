FUKKK.us

25 awesome web creations every month!

  • Suggest awesome web creations / demos / experiments to @fukkk_us
  • @fukkk_us shares these suggestions on Twitter
  • The community favorites the best web creations / demos / experiments
  • @fukkk_us curates a list of the top 25 at the end of each month
  • And you enjoy only the caziest stuff of the interwebs inside your mailbox!
  • Suggest awesome web demos to @fukkk_us and vote on Twitter to create the top 25 each month.
